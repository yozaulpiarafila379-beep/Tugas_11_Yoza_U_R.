1.Sintaks Dasar Redirecting Constructor

  • Pada program ini, redirecting constructor digunakan pada class Point untuk mengarahkan pemanggilan constructor ke constructor utama.
  • Constructor utama Point(this.x, this.y) digunakan untuk menginisialisasi nilai koordinat x dan y.
  • Constructor origin() digunakan untuk membuat titik pada koordinat (0,0).
  • Constructor onXAxis() digunakan untuk membuat titik pada sumbu X dengan nilai Y = 0.
  • Constructor onYAxis() digunakan untuk membuat titik pada sumbu Y dengan nilai X = 0.
  • Program kemudian menampilkan beberapa objek Point yang dibuat menggunakan redirecting constructor.

Kode Dart:


// ======================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI.2A
// ======================================

class Point {
  final double x;
  final double y;

  Point(this.x, this.y);

  Point.origin() : this(0, 0);

  Point.onXAxis(double x) : this(x, 0);

  Point.onYAxis(double y) : this(0, y);

  @override
  String toString() {
    return 'Point($x, $y)';
  }
}

void main() {
  var origin = Point.origin();
  var pointX = Point.onXAxis(5);
  var pointY = Point.onYAxis(3);

  print('Origin      : $origin');
  print('Point X     : $pointX');
  print('Point Y     : $pointY');
}

2.Masalah Tanpa Redirecting Constructor

  • Pada program ini, class Rectangle memiliki beberapa constructor yang menginisialisasi field secara terpisah.
  • Constructor square() digunakan untuk membuat objek persegi dengan panjang dan lebar yang sama.
  • Constructor withDefault() digunakan untuk membuat objek dengan ukuran bawaan.
  • Tanpa redirecting constructor, setiap constructor harus mengisi field satu per satu sehingga terjadi pengulangan kode.
  • Semakin banyak constructor yang dibuat, semakin sulit kode untuk dipelihara.
  • Program kemudian menampilkan data objek yang dibuat tanpa redirecting constructor.

Kode Dart:


// ======================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI.2A
// ======================================

class Rectangle {
  final double width;
  final double height;

  Rectangle(this.width, this.height);

  Rectangle.square(double side)
      : width = side,
        height = side;

  Rectangle.withDefault()
      : width = 100,
        height = 50;
}

void main() {
  var square = Rectangle.square(10);
  var defaultRect = Rectangle.withDefault();

  print("=== DATA PERSEGI ===");
  print("Width  : ${square.width}");
  print("Height : ${square.height}");

  print("\n=== DATA DEFAULT ===");
  print("Width  : ${defaultRect.width}");
  print("Height : ${defaultRect.height}");
}

3.Solusi dengan Redirecting Constructor

  • Pada program ini digunakan redirecting constructor untuk mengarahkan constructor lain ke constructor utama.
  • Constructor square() digunakan untuk membuat objek persegi dengan ukuran yang sama.
  • Constructor withDefault() digunakan untuk membuat objek dengan ukuran standar.
  • Constructor fromWidth() digunakan untuk membuat objek berdasarkan nilai lebar tertentu.
  • Redirecting constructor membantu mengurangi duplikasi kode dan membuat program lebih rapi.
  • Program kemudian menampilkan beberapa objek Rectangle yang dibuat menggunakan redirecting constructor.

Kode Dart:


// ======================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI.2A
// ======================================

class Rectangle {
  final double width;
  final double height;

  Rectangle(this.width, this.height);

  Rectangle.square(double side)
      : this(side, side);

  Rectangle.withDefault()
      : this(100, 50);

  Rectangle.fromWidth(double width)
      : this(width, width * 0.618);

  @override
  String toString() {
    return 'Rectangle(width: $width, height: $height)';
  }
}

void main() {
  var square = Rectangle.square(10);
  var defaultRect = Rectangle.withDefault();
  var goldenRect = Rectangle.fromWidth(100);

  print("=== DATA PERSEGI ===");
  print(square);

  print("\n=== DATA DEFAULT ===");
  print(defaultRect);

  print("\n=== GOLDEN RATIO RECTANGLE ===");
  print(goldenRect);
}

4.Redirecting Constructor dengan Named Parameters

  • Pada program ini, redirecting constructor digunakan bersama named parameters pada class Person.
  • Constructor utama menerima parameter name, age, dan email.
  • Constructor anonymous() digunakan untuk membuat objek dengan nilai default.
  • Constructor fromName() digunakan untuk membuat objek hanya berdasarkan nama.
  • Constructor adult() digunakan untuk membuat objek dengan usia dewasa secara otomatis.
  • Constructor child() digunakan untuk membuat objek anak dengan usia tertentu.
  • Program kemudian menampilkan data beberapa objek Person yang dibuat menggunakan redirecting constructor.

Kode Dart:


// ======================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI.2A
// ======================================

class Person {
  final String name;
  final int age;
  final String? email;

  Person({
    required this.name,
    required this.age,
    this.email,
  });

  Person.anonymous()
      : this(name: 'Anonymous', age: 0);

  Person.fromName(String name)
      : this(name: name, age: 0);

  Person.adult(String name)
      : this(name: name, age: 18);

  Person.child(String name, int years)
      : this(
          name: name,
          age: years,
          email: null,
        );

  @override
  String toString() {
    return 'Name: $name, Age: $age, Email: ${email ?? "N/A"}';
  }
}

void main() {
  print(Person.anonymous());
  print(Person.fromName("Yoza"));
  print(Person.adult("Rafila"));
  print(Person.child("Budi", 10));
}

5.Praktik 1 – Configuration Class

  • Pada program ini dibuat class DatabaseConfig untuk menyimpan konfigurasi database.
  • Constructor utama digunakan untuk menerima seluruh konfigurasi yang diperlukan.
  • Redirecting constructor digunakan untuk membuat konfigurasi development, production, dan localhost.
  • Dengan cara ini pembuatan konfigurasi menjadi lebih sederhana dan konsisten.
  • Program kemudian menampilkan informasi konfigurasi yang digunakan.

class DatabaseConfig {
  final String host;
  final int port;
  final String database;

  DatabaseConfig({
    required this.host,
    required this.port,
    required this.database,
  });

  DatabaseConfig.development()
      : this(
          host: 'localhost',
          port: 5432,
          database: 'dev_db',
        );

  DatabaseConfig.production()
      : this(
          host: 'db.example.com',
          port: 5432,
          database: 'prod_db',
        );

  DatabaseConfig.localhost()
      : this(
          host: '127.0.0.1',
          port: 3306,
          database: 'local_db',
        );

  @override
  String toString() {
    return 'Host: $host, Port: $port, Database: $database';
  }
}

void main() {
  print(DatabaseConfig.development());
  print(DatabaseConfig.production());
  print(DatabaseConfig.localhost());
}

6.Redirecting Constructor dengan Validasi

  • Pada program ini digunakan factory constructor yang melakukan validasi data sebelum objek dibuat.
  • Class BankAccount digunakan untuk menyimpan informasi rekening bank berupa nomor akun, nama pemilik, dan saldo.
  • Factory constructor create() digunakan untuk memvalidasi data yang dimasukkan oleh pengguna.
  • Nomor akun dan nama pemilik tidak boleh kosong.
  • Saldo awal tidak boleh bernilai negatif.
  • Constructor fromJson() digunakan untuk membuat objek dari data JSON dengan memanfaatkan constructor create().
  • Program kemudian menampilkan informasi rekening yang berhasil dibuat setelah seluruh data lolos validasi.

Kode Dart:


// ======================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI.2A
// ======================================

class BankAccount {
  final String accountNumber;
  final String ownerName;
  final double balance;

  const BankAccount._({
    required this.accountNumber,
    required this.ownerName,
    required this.balance,
  });

  factory BankAccount.create({
    required String accountNumber,
    required String ownerName,
    double initialBalance = 0,
  }) {
    if (accountNumber.isEmpty) {
      throw ArgumentError(
        'Nomor akun tidak boleh kosong',
      );
    }

    if (ownerName.isEmpty) {
      throw ArgumentError(
        'Nama pemilik tidak boleh kosong',
      );
    }

    if (initialBalance < 0) {
      throw ArgumentError(
        'Saldo awal tidak boleh negatif',
      );
    }

    return BankAccount._(
      accountNumber: accountNumber,
      ownerName: ownerName,
      balance: initialBalance,
    );
  }

  factory BankAccount.fromJson(
    Map json,
  ) {
    return BankAccount.create(
      accountNumber:
          json['accountNumber'] ?? '',
      ownerName:
          json['ownerName'] ?? '',
      initialBalance:
          (json['balance'] ?? 0).toDouble(),
    );
  }

  @override
  String toString() {
    return 'Nomor Akun: $accountNumber, '
        'Pemilik: $ownerName, '
        'Saldo: Rp${balance.toStringAsFixed(2)}';
  }
}

void main() {
  var account = BankAccount.create(
    accountNumber: '123456789',
    ownerName: 'Yoza Ulpia Rafila',
    initialBalance: 500000,
  );

  print(account);

  var jsonAccount = BankAccount.fromJson({
    'accountNumber': '987654321',
    'ownerName': 'Budi',
    'balance': 750000,
  });

  print(jsonAccount);
}

7.Multiple Redirecting Levels

  • Pada program ini, redirecting constructor digunakan untuk mengarahkan proses pembuatan objek ke constructor utama.
  • Class Product memiliki beberapa constructor yaitu basic, withCategory, dan full.
  • Constructor basic digunakan untuk membuat produk dengan data minimum.
  • Constructor withCategory digunakan untuk membuat produk dengan tambahan informasi kategori.
  • Constructor full digunakan untuk membuat produk dengan informasi yang lebih lengkap seperti stok dan tanggal kedaluwarsa.
  • Seluruh constructor tersebut mengarahkan proses inisialisasi ke constructor utama sehingga mengurangi duplikasi kode.
  • Program kemudian menampilkan informasi produk yang dibuat menggunakan masing-masing constructor.

Kode Dart:


// ==========================================
// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A
// ==========================================

class Product {
  final String id;
  final String name;
  final double price;
  final String category;
  final int stock;
  final DateTime? expiryDate;

  Product({
    required this.id,
    required this.name,
    required this.price,
    required this.category,
    required this.stock,
    this.expiryDate,
  });

  Product.basic(String id, String name, double price)
      : this(
          id: id,
          name: name,
          price: price,
          category: 'General',
          stock: 0,
        );

  Product.withCategory(
    String id,
    String name,
    double price,
    String category,
  ) : this(
          id: id,
          name: name,
          price: price,
          category: category,
          stock: 0,
        );

  Product.full({
    required String id,
    required String name,
    required double price,
    required String category,
    required int stock,
    DateTime? expiry,
  }) : this(
          id: id,
          name: name,
          price: price,
          category: category,
          stock: stock,
          expiryDate: expiry,
        );
}

void main() {
  var basic = Product.basic('P001', 'Item 1', 10.0);

  var categorized = Product.withCategory(
    'P002',
    'Item 2',
    20.0,
    'Electronics',
  );

  var full = Product.full(
    id: 'P003',
    name: 'Item 3',
    price: 30.0,
    category: 'Food',
    stock: 100,
    expiry: DateTime.now().add(Duration(days: 30)),
  );

  print('=== Product Basic ===');
  print('${basic.id} - ${basic.name}');
  print('');

  print('=== Product Category ===');
  print('${categorized.id} - ${categorized.category}');
  print('');

  print('=== Product Full ===');
  print('${full.id} - ${full.name}');
  print('Stock : ${full.stock}');
  print('Category : ${full.category}');
}
